# Amazon PPC Wiki — Student Edition

A beginner-friendly knowledge base for learning Amazon PPC advertising from scratch.

Built for students, aspiring Amazon advertising specialists, and anyone who wants to understand how Amazon PPC works — without the agency jargon or internal playbooks.

**Complete beginners:** Start at [Section 1: Foundations](sections/01-1-foundations-and-fundamentals/index.md) and work forward.

**Have some experience?** Jump to the section you need:

## What's inside

- **25 topic sections** covering every aspect of Amazon PPC
- **120+ pages** of beginner-friendly explanations with real examples
- **Glossary** of every PPC term and acronym
- **Formulas and calculators** you'll use on the job
- **Case studies** with real campaign data
- **Career guidance** including certification paths

## What's NOT here

This is a student-only resource. Agency operations, client communication playbooks, team management guides, and internal SOPs have been removed. This wiki is purely educational.

## Quick start

Read online: **[projectamazonph.github.io/Amazon-PPC-Student-Wiki](https://projectamazonph.github.io/Amazon-PPC-Student-Wiki/)**

Read offline:

```bash
git clone https://github.com/projectamazonph/Amazon-PPC-Student-Wiki.git
cd Amazon-PPC-Student-Wiki
pip install -r requirements.txt
mkdocs serve
```

Open the local URL printed in your terminal.
